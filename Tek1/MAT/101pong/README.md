# 101pong

### Compilation:
```
$ pip3 install numpy (si pas installé)
$ make
```

### Execution:
```
$ ./101pong 1 3 5 7 9 -2 4
```

## Informations sur le projet:

    Basé sur des calculs de vecteurs, permet de connaître la position de la balle à un instant T donné.
