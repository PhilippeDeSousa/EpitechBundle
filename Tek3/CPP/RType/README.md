# RType

### Compilation:
```
This requires Conan
$ ./compile.sh
```

### Execution (two terminals):
```
$ ./build/Server/bin/server [port]
```
```
$ ./build/Client/bin/client 127.0.0.1 [port]
```

## Informations:

    Goal of the project:
    Recreate a RType game in C++ using a proper Game Engine.
